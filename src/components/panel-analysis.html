준설량 모의 계산

01. 측량 성과 업로드

첨부할 파일을 여기에 끌어다 놓거나.
파일 업로드 버튼을 직접 선택해 주세요.

  • 준설 전 측량자료가 있는 경우 업로드하세요. 없다면 24년 4월 측량된 자료를 기반으로 준설량을 계산합니다.

02. 측량 성과 업로드

첨부할 파일을 여기에 끌어다 놓거나.

  • 파일은 SHP 파일 형식을 지원합니다.
  • 업로드 용량은 50Mb 이하만 가능합니다.
